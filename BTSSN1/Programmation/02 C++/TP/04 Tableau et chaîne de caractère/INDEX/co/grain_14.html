Exercice 3 : On souhaite récupérer une adresse IP entré par un utilisateur.

Une solution est de lire la chaîne de caractères complète puis de l'analyser caractère après caractère.

Dans ce cas toute erreur de l'utilisateur sera effectivement détectée et aucune exécution incorrecte ne sera possible.

Le programme testera chaque caractère afin de trouver les séparateurs éventuels (caractère autre qu'un chiffre) puis transformera le texte précédent ce séparateur en nombre entier.

Pour réaliser ce programme, vous pourrez utiliser les fonctions suivantes :

#include <cstring>

cin.getline permet de lire une ligne entrée au clavier jusqu'à un maximum de n caractères (2nd argument) et la met dans une chaîne de caractères (1er argument).

  • Exemple : char chaine [10]; cin.getline (chaine, 10);

strncpy copie les n (3ème argument) premiers octets de la chaîne src (2eme argument) (y compris le caractère `\0' final) dans la chaîne dest (1er argument).

  • Exemple : strncpy (dest, src+2, 4); si src contient : "abcdefghij", alors dest contient : "cdef"

#include <cstdlib>

atoi retourne un entier correspondant à la chaîne de caractères passé en argument.

  • Exemple : char chaine [10] = "123"; int entier; entier = atoi (chaine); // entier contient la valeur entière 123

strlen retourne la longueur de la chaîne de caractères passé en argument.

  • Exemple : char chaine [10] = "123456"; int longueur; longueur = strlen (chaine); // donne 6

Étape 1

Entrée de l'adresse IP :

Écrire un programme demandant l'adresse IP sous la forme xxx.xxx.xxx.xxx, la stocker dans une chaîne de 30 caractères et rechercher le premier caractère de cette chaîne qui n'est pas un chiffre (recherche du premier séparateur).

Indice : utiliser la table du code ascii.

Étape 2

Vérification des différents champs

Modifier le programme pour contrôler que l'adresse IP est correcte.

  • Recherche des séparateurs et conversion de l'adresse :

    • Modifier le programme pour qu'il recherche la position des 3 séparateurs ( indice nous pourrons utiliser un tableau sep de 3 entier pour stocker les 3 positions )

    • Copie successivement les 4 parties (de 0 au 1er séparateur, du 1er séparateur au 2nd séparateur, du 2nd séparateur au 3ème et du 3ème au dernier octet de la chaîne dans des chaines octet1, octet2 etc...) et les convertir en entier à l'aide de la fonction atoi puis nous afficherons les entiers obtenus.

      Ex : pour 123.240.230.254 copier "123" dans une chaine octet1, "240" dans une chaine octet2, "230" dans une chaine octet3, "254" dans une chaine octet4 puis convertir ces chaînes en entier puis les afficher.

  • Contrôle de tous les caractères et conversion de l'adresse ( test de la robustesse du programme ):

    Modifier le programme pour qu'il affiche les 4 parties de l'adresse IP et vérifie qu'aucune erreur n'a été commise par l'utilisateur, c'est-à-dire :

    • les séparateurs sont bien des points ;

    • les nombres entre les séparateurs sont compris entre 0 et 255 ;

    • il n'y a pas d'autre caractère après le dernier nombre.

L'adresse IP sera lue dans une chaîne de caractères puis analysée par le programme.

  • Proposer des options possibles : indiquer la classe de l'adresse ou donner le masque correspondant etc...
