2. Vous crééerez une classe étudiant: 

Attributs: 

	Nom : de type string
	Prenom : de type string
	CNE : de type int
	notes : un tableau de réel contenant cinq cases ou chacune représente une note.

Methodes : 

	getMaxNote() : retourne la plus grande note de l’étudiant.
	getMinNote() : retourne la plus petite note de l’étudiant.
	getMoyenneNote() : retourne la moyenne des notes de l’étudiant.
	affiche() : permet d’afficher une description d’un étudiant y compris la plus grande et la plus petite note, la moyenne, nom, prénom, et le CNE.
	

utiliser le constructeur par defaut pour la création de vos objets.
