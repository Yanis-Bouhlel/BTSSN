Exercice 1 : Vérification des notions de bases

#include <iostream>

int main() {

int i ;

for (i = 0 ; i < 10 ; i = i + 1)

{

cout<<"iteration "<< i ;

}

cout<<"valeur de i apres la boucle "<< i <<endl ;

return 0 ;

}

1. Tester l'exemple çi-dessus !

2. Modifiez le programme afin de faire 100 itérations au lieu de 10. Testez.

Amélioration du programme 2.

2b. Modifier votre programme  afin qu'il demande à l'utilisateur“ Voulez-vous afficher à nouveau cette boucle? ” tant que celui-ci lui répond “ oui ”.
