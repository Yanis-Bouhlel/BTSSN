Pourquoi un serveur Web ?

Une entreprise a besoin de proposer à ses clients des pages web leur présentant les produits, les services, ...

Nous allons voir comment installer sur un serveur, les logiciels pour offrir ces fonctionnalités.

Pour un site WEB de base, un serveur Web propose des pages de type HTML. Mais ces pages sont statiques, c'est-à-dire que si on veut modifier les prix par exemple des produits ou leur référence, il faut modifier les valeurs dans les fichiers HTML. Cela est vraiment peu pratique donc les valeurs sont stockées dans une base de données et les pages sont dynamiques. Elles sont modifiées en fonction de la base de données.

Les pages dynamiques utilisent généralement 2 techniques différentes :

  • PHP (Hypertext Preprocessor) est un langage de programmation libre ;

  • ASP.NET est une technique pour générer à la demande des pages web. Le moteur d'ASP.NET peut être utilisé avec n'importe quel langage de programmation pour la plate-forme .NET (Visual Basic .NET, C#, JScript4...).

Nous nous limiterons à l'utilisation du PHP.
