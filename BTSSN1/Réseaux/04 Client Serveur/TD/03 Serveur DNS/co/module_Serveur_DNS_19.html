Exercice

Complétez les lignes suivantes du fichier "/etc/bind/named.conf.local" pour la recherche inverse (même configuration que dans l'exercice précédent) :

zone "16.172.in-addr.arpa" {

type master;

file "/etc/bind/db.172.16";

};

Complétez les lignes suivantes du fichier de configuration pour les trois adresses de l'exercice précédent :

2 PTR ns1.LJR.lan. ; serveur DNS

3 PTR www.LJR.lan. ; serveur WEB

254 PTR routeur ; routeur
