Exercice

Complétez le fichier /etc/bind/db.LJR.fr qui gère la zone LJR.lan sachant que :

  • le nom du serveur de DNS est ns1 et son adresse IP 173.87.0.2 ;

  • le serveur Web a pour nom www et son adresse IP est 173.87.0.3 ;

  • le routeur (passerelle par défaut) a pour nom routeur et son adresse IP est 173.87.0.254 ;

  • l'adresse mail du responsable de la maintenance est root.LJR.lan.

@ IN SOA ns1.LJR.lan. root.LJR.lan. (

2016021301 ; se = serial number

172800 ; ref = refresh = 2d

900 ; ret = update retry = 15m

1209600 ; ex = expiry = 2w

3600 ; min = minimum = 1h

);

@ IN NS ns1.LJR.lan.

ns1 A 173.87.0.2 ; serveur DNS

www A 173.87.0.3 ; serveur WEB

routeur A 173.87.0.254 ; routeur
