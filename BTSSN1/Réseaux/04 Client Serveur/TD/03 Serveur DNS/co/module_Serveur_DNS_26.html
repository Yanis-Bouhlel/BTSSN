DNS est l'acronyme de Domain Name Service.

La combinaison du nom de machine et de son nom de domaine (exemple : mail.ac-creteil.fr) est le Fully Qualified Domain Name.

L'extension en premier (en premier en partant de la droite) s'appelle le Top Level Domain.

fr est un domaine et google.fr est un sous-domaine de fr.

Le serveur DNS permet d'avoir l'adresse IP à partir du FQDN. Il s'agit dans ce cas d'une recherche directe.

Dans le cas contraire (recherche inverse), on obtient le FQDN à partir de l'adresse IP.

La commande pour tester le serveur DNS est nslookup.

DDNS est l'acronyme de Dynamic Domain Name Service.

Le service DDNS est un service utilisé pour mettre à jour automatiquement un serveur DNS lors de l'attribution dynamique d'une adresse IP par un serveur DHCP.
