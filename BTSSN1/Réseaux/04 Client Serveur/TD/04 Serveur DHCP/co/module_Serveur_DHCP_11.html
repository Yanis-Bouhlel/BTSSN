Exercice : Fichier de configuration sous Linux

Complétez les lignes suivantes du fichier « /etc/dhcp/dhcpd.conf » pour le réseau 195.32.10.0/24 sachant que l'adresse de la passerelle est 195.32.10.254. Le domaine est « snir.lan » et le serveur est 195.32.10.1. Les adresses de 195.32.10.50 à 195.32.10.159 sont réservées pour le serveur DHCP.

subnet 195.32.10.0 netmask 255.255.255.0 {

range 195.32.10.50 195.32.10.159

option routers 195.32.10.254

option domain-name snir.lan

option domain-name-servers 195.32.10.1

}
