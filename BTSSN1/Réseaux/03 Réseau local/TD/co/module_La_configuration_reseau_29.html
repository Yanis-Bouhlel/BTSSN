Notation CIDR

Une forme plus courte pour préciser l'adresse réseau et le masque est la « notation CIDR » (Classless Inter-Domain Routing ou routage sans classes entre domaines). Elle donne l'adresse du réseau suivie par une barre oblique (ou slash, « / ») et le nombre de bits à 1 dans la notation binaire du masque de sous-réseau (préfixe).

Si le masque comprend n bits à 1, la valeur à écrire est n.

Exemple : Notation CIDR pour les classes

Le réseau 13.0.0.0 (classe A) a un masque 255.0.0.0 → 13.0.0.0/8 (8 bits à 1 dans le masque)

Le réseau 146.22.0.0 (classe B) a un masque 255.255.0.0 → 146.22.0.0/16 (16 bits à 1 dans le masque)

Le réseau 202.12.93.0 (classe C) a un masque 255.255.255.0 → 202.12.93.0/24 (24 bits à 1 dans le masque)
