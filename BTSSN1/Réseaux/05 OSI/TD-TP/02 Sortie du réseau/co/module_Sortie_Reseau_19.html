Routage

Définition :

Le routage est le mécanisme par lequel des chemins sont sélectionnés dans un réseau pour acheminer les données d'un expéditeur jusqu'à un ou plusieurs destinataires.

Le routage sera étudié dans un autre cours mais nous devons en voir quelques aspects pour les cas où plusieurs réseaux sont interconnectés.

Définition : Routeur

Un routeur est un système avec plusieurs interfaces réseaux, une par réseau donc une adresse IP :

Maintenant ajoutons un autre réseau :

La passerelle par défaut pour les ordinateurs PC4 et PC5 est 192.168.3.254.

Supposons que PC2 veut envoyer une trame à PC4. Il va l'envoyer à sa passerelle par défaut (192.168.2.254), c'est-à-dire le routeur 0. Or le routeur 0 ne connaît que deux réseaux et donc ne sait pas comment atteindre le réseau 192.168.3.0/24. La trame n'atteindra pas l'ordinateur PC4.

Il faut donc configurer le routeur 0 pour qu'il connaisse le réseau 192.168.3.0/24, de même il faut configurer le routeur 1 pour qu'il connaisse le réseau 192.168.1.0/24.
