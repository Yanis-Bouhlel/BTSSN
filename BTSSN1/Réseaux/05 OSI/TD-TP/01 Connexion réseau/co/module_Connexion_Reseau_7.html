Format de l'adresse MAC

Les adresses MAC sont composées de 6 octets :

  • 3 octets qui font référence au constructeur de la carte de réseau. Ils constituent l'OUI (Organizationaly Unique Identifier). Chaque constructeur possède une référence unique qui est attribuée par l'IEEE.

  • 3 octets pour le code de la carte.

Il existe une adresse particulière appelée adresse de diffusion (broadcast en anglais). Cette adresse est utilisée pour s'adresser à tous les ordinateurs du réseau. Sa valeur est (notation en hexadécimal) FF:FF:F:FF:FF:FF.

Exemple : Exemples de code fabricant ≪ Organizationnally Unique Identifier ≫ (OUI)

00:00:0C : Cisco

00:03:93 : Apple

02:80:8C : 3Com

08:00:20 : Sun

08:00:5A : IBM

00:30:6C : HP

00:05:5D : D-link

00:07:CB : Freebox

Complément :

Les deux bits du poids faible du premier octet des adresses MAC sont des bits particuliers :

  • I/G : 0 : adresse individuelle (unicast) / 1 : adresse de groupe (multicast)

  • U/L : 0 : adresse universelle (fournie par l'IEEE) / 1 : adresse locale (fixée par administrateur)

Combinaison des bits I/G et U/L :

  • I/G = 0 et U/L : 0 pour unicast,

  • I/G = 0 et U/L : 1 pour multicast,

  • I/G = 1 et U/L : 1 pour broadcast.

L'adresse de destination peut être unique (Unicast), de groupe (Multicast) ou de diffusion générale (Broadcast = FF:FF:FF:FF:FF:FF).

Dans une adresse de groupe, le premier bit transmis est à 1. Si les autres bits ne changent pas, l'adresse de groupe correspond à toutes les cartes d'un même constructeur.
