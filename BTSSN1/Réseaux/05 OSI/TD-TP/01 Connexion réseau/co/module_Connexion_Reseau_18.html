Commande arp

La commande arp s'appuie sur le protocole ARP. Elle permet de gérer la table ARP.

Syntaxe

arp –a pour afficher le cache ARP

arp –d @IP pour supprimer un couple @ip-@MAC du cache ARP (Si @IP = *, on les supprime tous)

arp –s @IP @MAC pour insérer un couple d'adresses IP-MAC dans le cache ARP

Exemple : Commande arp -a
