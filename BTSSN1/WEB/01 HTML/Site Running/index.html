Qu'es que le running ?

Le running est une pratique libre de la course à pied, accompagnée d'objectifs propres à ceux d'une discipline. Le running s'inscrit dans le culte de la performance : courir plus longtemps ou plus vite par exemple. ... Running est un anglicisme, qui vient du mot courir.

les bienfaits du running

  • Renforcement physique
  • Perte de poids
  • Santé cardiaque
  • La course à pied aide à prévenir l'hypertension artérielle
  • Réduire les effets du diabète
  • Atténuer les effets de l'asthme
  • Renforcement des articuation et stabiluté
  • Auto-contrôle
