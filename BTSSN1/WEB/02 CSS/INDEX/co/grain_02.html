En suivant les étapes vous arriverez à structure page web suivante:

1 . Titre (balise <title>)

→ Créer dans votre dossier cave, la page d'accueil index.html suivante :

→Écrire : SFCV : Spécialiste Français n°1 des Caves à Vin à la place de TODO0. Cela sera donc le titre de votre page.

2 . Le bloc central( div bloc ):

→ Modifier le document index.html (dans le cadre bloc <!--TODO3-->) afin qu'il reproduise la figure ci-dessous.

Vous supprimerez donc <!--TODO3--> et vous écrirez dans son emplacement.

On tiendra compte des indications suivantes :

  • l'en-tête « Spécialiste français n°1 des caves à vin » est de type <h2>

  • on utilisera des paragraphes (<p>) dont un utilise une liste numérotée (<ol>).

Attention : vous avez tendance à « abuser » des balises <br> pour respecter la mise en page demandée.

Ne faites pas ça ! Et conservez seulement les sauts de lignes nécessaires dans le texte d'un paragraphe.

On vous rappelle que la présentation du document à l'écran sera réalisée à la fin par l'application d'un style (CSS). Pour l'instant, consacrez vous à la structure de vos pages plus qu'à leur visualisation.

3 . Les liens absolus et relatifs (balise <a>)

→ Modifier le document index.html (dans le cadre bloc <!--TODO3--> et pied <!--TODO4-->) afin qu'il reproduise la figure ci-dessous.

On tiendra compte des indications suivantes :

  • l'en-tête « Liens » est de type <h2>

  • on déclarera deux paragraphes pour chaque lien créé

  • le lien "Mentions légales" pointera sur un document mentions.html

4 . Les images (balise <img>)

→ Modifier le document index.html (dans le cadre entete <!--TODO1-->) afin qu'il reproduise la figure ci-dessous.

On tiendra compte des indications suivantes :

  • l'en-tête pour le nom de la société SFCV sera de type <h1>

  • on utilisera un autre en-tête de type <h3> pour " N°1 des ... "

  • le chemin du fichier image est : images/logo.png (on ajoute un texte alternatif pour cette image)

5 . Le menu

→ Modifier le document index.html (dans le cadre menu <!--TODO2-->) afin qu'il reproduise la figure ci-dessous.

On tiendra compte des indications suivantes pour créer le menu de gauche :

  • on utilisera une liste non numérotée de type <ul> donc chaque élément de liste <li> contiendra un élément (lien) du menu

  • le lien " Accueil " pointera sur un document index.html ( il faudra créer ce fichier dans votre dossier cave )

  • le lien " Produits " pointera sur un document produits.html ( il faudra créer ce fichier dans votre dossier cave )

  • le lien " Services " pointera sur un document services.html ( il faudra créer ce fichier dans votre dossier cave )

  • le lien " Contact " pointera sur un document contact.html ( il faudra créer ce fichier dans votre dossier cave )

Remarque : ces choix correspondent au menu « classique » d'une société commerciale.

6 . Validation

→ Vous devez valider la page index.html en utilisant le service en ligne du W3C :

https://validator.w3.org/#validate_by_upload

Puis, tester les liens externes.

Lire l'annexe 3 sur la DTD HTML 4.01 Strict pour comprendre le service réalisé par le validateur.
